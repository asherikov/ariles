ariles::bridge::rapidjson::BasicIStreamWrapper< StreamType > Class Template Reference

Detailed Description

template<typename StreamType>
class ariles::bridge::rapidjson::BasicIStreamWrapper< StreamType >

Wrapper of std::basic_istream into RapidJSON's Stream concept.

The classes can be wrapped including but not limited to:

  • std::istringstream
  • std::stringstream
  • std::wistringstream
  • std::wstringstream
  • std::ifstream
  • std::fstream
  • std::wifstream
  • std::wfstream
